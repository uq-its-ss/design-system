# `@uqds/accordion`

> Accordion component

## Usage

### Installation

With Yarn:

```shell
yarn add @uqds/accordion
```

With NPM:

```shell
npm i -S @uqds/accordion
```

Installing an alpha version

```shell
npm i -S @uqds/accordion@latest --tag=alpha
```

---

@uqds/accordion
