# `@uqds/header-legacy`

> Header component

## Usage

### Installation

With Yarn:
```shell
yarn add @uqds/header-legacy
```

With NPM:
```shell
npm i @uqds/header-legacy
```

---

## Dependency graph

├─ @uqds/grid
│  └─ @uqds/core
