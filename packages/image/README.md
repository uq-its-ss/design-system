# `@uqds/layout`

> layout component

## Usage

### Installation

With Yarn:

```shell
yarn add @uqds/layout
```

With NPM:

```shell
npm i @uqds/layout
```

### CSS

Sass:

```scss
@import "@uqds/layout/src/scss/main.scss";
```

Postcss:

```css
@import "@uqds/layout";
```

---

## Dependency graph

```shell
@uqds/layout
├─ @uqds/core
```
