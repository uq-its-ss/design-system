# `@uqds/navigation`

> Navigation components

## Usage

### Installation

With Yarn:

```shell
yarn add @uqds/navigation
```

With NPM:

```shell
npm i @uqds/navigation
```

### CSS

Sass:

```scss
@import "@uqds/navigation/src/scss/main.scss";
```

Postcss:

```css
@import "@uqds/navigation";
```

---

## Dependency graph

```shell
@uqds/navigation
├─ @uqds/core
└─ @uqds/icon
   └─ @uqds/core
```
