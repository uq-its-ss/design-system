# `@uqds/pill`

> pill component

## Usage

### Installation

With Yarn:

```shell
yarn add @uqds/pill
```

With NPM:

```shell
npm i @uqds/pill
```

### CSS

Sass:

```scss
@import "@uqds/pill/src/scss/main.scss";
```

Postcss:

```css
@import "@uqds/pill";
```

---

## Dependency graph

```shell
@uqds/pill
├─ @uqds/core
```
