# `@uqds/react`

> React components

## Usage

### Installation

With Yarn:

```shell
yarn add @uqds/react
```

With NPM:

```shell
npm i -S @uqds/react
```

Installing an alpha version. NPM won't install alpha versions unless you explicity ask for it with `--tag`

```shell
npm i -S @uqds/react@latest --tag=alpha
```
