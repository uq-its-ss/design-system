# `@uqds/site-header`

> site header component

## Usage

### Installation

With Yarn:
```shell
yarn add @uqds/site-header
```

With NPM:
```shell
npm i @uqds/site-header
```

---

## Dependency graph

├─ @uqds/grid
│  └─ @uqds/core
