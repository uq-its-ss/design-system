# `@uqds/social`

> social component

## Usage

### Installation

With Yarn:

```shell
yarn add @uqds/social
```

With NPM:

```shell
npm i @uqds/social
```

### CSS

Sass:

```scss
@import "@uqds/social/src/scss/main.scss";
```

Postcss:

```css
@import "@uqds/social";
```

---

## Dependency graph

```shell
@uqds/social
├─ @uqds/core
```
