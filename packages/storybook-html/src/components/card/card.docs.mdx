## Overview

The card component is used to provide a brief summary of content or a task, often with a link to more detail. Cards are frequently displayed alongside other cards to group related content or tasks.

## Default example

### Card default

The most common and basic card used.

<Canvas withSource="none">
  <Story id="components-cards--card-default" />
</Canvas>

```html
<div class="card">
  <div class="card__header"></div>
  <div class="card__body">
    <a class="card__link" href="#">
      <div class="card__image-container" style="background-image: url('https://campuses.uq.edu.au/files/9749/gatton-card2.jpg');"></div>
    </a>
    <div class="card__content">
      <h3 class="card__title">
        <a class="card__link" href="#">Title of item</a>
      </h3>
      <div class="card__summary">Lorem ipsum dolor sit amet, consetetur sadipscing elitr, sed diam nonumy eirmod
        tempor invidunt ut labore et dolore magna aliquyam erat, sed diam voluptua.
      </div>
    </div>
  </div>
  <div class="card__footer"></div>
</div>
```

## Variant examples

### Card compact

Slightly smaller version of a card. Typically used with just a title and no additional information. <code>card--compact</code> 

<Canvas withSource="none">
  <Story id="components-cards--card-compact" />
</Canvas>

```html
<div class="card card--compact">
  ...
  ...
</div>
```

### Card with action links

Add action links to cards when multiple link options are required. Link for main title can also be removed. 

<Canvas withSource="none">
  <Story id="components-cards--card-with-links" />
</Canvas>

```html
<div class="card">
  ...
  ...
  <div class="card__footer">
    <div class="uq-grid">
      <div class="uq-grid__col uq-grid__col--3">
        <a class="card__link" href="#">Action link A</a>
      </div>
      <div class="uq-grid__col uq-grid__col--3">
        <a class="card__link" href="#">Action link B</a>
      </div>
    </div>
  </div>
</div>
```
