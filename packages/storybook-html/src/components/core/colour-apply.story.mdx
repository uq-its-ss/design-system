# Applying colour

## Principles

The Design System specifies colour based on the [UQ Brand Guidelines](https://marketing-communication.uq.edu.au/one-brand) in combination with accessibility standards and user interface standards.

Colour should be applied to user interfaces in only two ways:

- to apply the UQ brand to the user interface
-	in functional and semantic ways such as containers for related content or red for errors. 

Colour should not be added to an interface for the sole purpose of creating visual interest. Standard styles and components will incorporate appropriate colour. When a user interface needs to be more visually engaging to achieve its goals, incorporate photography aligned with the brand guidelines.


## Accessibility

Use of colour must meet the following requirements:

-	[sufficient colour contrast](https://www.w3.org/WAI/WCAG21/Understanding/contrast-minimum) between text and background colours
-	[colour is not used as the only visual means of conveying information](https://www.w3.org/WAI/WCAG21/Understanding/use-of-color).


## UQ Purple and purple gradient

UQ purple should be the only accent colour on a screen. It is already incorporated via the header and footer, typography, icons and other components. Additional use of UQ purple should be thoughtful and sparing. 

The purple gradient is already incorporated in the header, footer and other components. It should not be used elsewhere in user interfaces without consultation.
