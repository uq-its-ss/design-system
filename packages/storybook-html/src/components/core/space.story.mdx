## Layout spacing

The layout scale is used for arranging components and other UI parts into a full page layout. It has larger increments that are used to control the density of a design. Use the smaller layout tokens to create more dense compositions and direct relationships. Use the larger tokens to increase the amount of white space and to disassociate sections.

## Applying spacing

Consideration should be given to how spacing is applied within components and the page.

### Container space

Applies padding to containers

- For all user interface containers.
- Apply padding on all sides. 

<table>
  <tr>
    <th>Type</th>
    <th>Example</th>
    <th>Usage guide</th>
  </tr>
  <tr>
    <td>Container</td>
    <td><div class="uqds-container uqds-container-space"><code>$space;</code></div></td>
    <td>
      <code>padding:$space;</code>
      <ul>
        <li>For all user interface containers.</li>
        <li>Apply padding equally on all sides.</li>
      </ul>
    </td>
  </tr>
  <tr>
    <td>Container size</td>
    <td><div class="uqds-container uqds-container-space-lg"><code>$space-lg;</code></div></td>
    <td>
      <code>padding:$space-lg;</code>
      <ul>
        <li>adjust space using available space scale variables</li>
        <li>Space increments are used to control the density</li>
      </ul>
    </td>
  </tr>
  <tr>
    <td>Compressed</td>
    <td><div class="uqds-container uqds-compressed-space-lg"><code>$space-compressed-lg $space-lg;</code></div></td>
    <td>
      <code>paddding:$space-compressed-lg $space-lg;</code>
      <ul>
        <li>Reduces the top and bottom padding</li>
        <li>Creates better spacing balance for cell-like containers</li>
        <li>Increases the density of grouped content</li>
        <li>Examples include, data tables, list group items, buttons and pills.</li>
      </ul>
    </td>
  </tr>
  <tr>
    <td>Stretched</td>
    <td><div class="uqds-container uqds-stretched-space"><code>$space-stretched $space;</code></div></td>
    <td>
      <code>paddding:$space-stretched $space;</code>
      <ul>
        <li>increases the top and bottom padding</li>
        <li>Creates better spacing balance for long or stretched objects</li>
        <li>Examples include, textboxes, textareas, and other form elements</li>
      </ul>
    </td>
  </tr>
</table>

### Stacked spacing

Stacks content vertically, applying top margin. 

- allows us to target the objects space through the css cascade, and the space to the next object. 
- use For all stacked content. So for example panels, form fields and anything else that is stacked vertically.
