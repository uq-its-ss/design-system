# Typography

Typography is the art and technique of arranging type (letters and characters). It sets the theme and mood of the design via the typeface family and font styles like size, weight, style, colour, line-height, letter-spacing etc on single and multiple lines.

## Font list

The 3 typefaces used in the design system are Roboto, Merriweather & Montserrat.

## Font stack

Roboto
```scss
font-family: "Roboto", "Helvetica Neue", Helvetica, Arial, sans-serif;
```
Merriweather
```scss
font-family: Merriweather, Georgia, serif;
```
Montserrat
```scss
font-family: Montserrat, Helvetica, Arial, sans-serif;
```

## Loading fonts

Copy the HTML or CSS lines below and paste it into your project to load design system fonts.

### HTML
```html
<link rel="stylesheet" type="text/css" href="https://static.uq.net.au/v13/fonts/Roboto/roboto.css" />
<link rel="stylesheet" type="text/css" href="https://static.uq.net.au/v13/fonts/Merriweather/merriweather.css" />
<link rel="stylesheet" type="text/css" href="https://static.uq.net.au/v13/fonts/Montserrat/montserrat.css" />
```

### CSS/Sass
```css
@import url('https://static.uq.net.au/v13/fonts/Roboto/roboto.css');
@import url('https://static.uq.net.au/v13/fonts/Merriweather/merriweather.css');
@import url('https://static.uq.net.au/v13/fonts/Montserrat/montserrat.css');
```
