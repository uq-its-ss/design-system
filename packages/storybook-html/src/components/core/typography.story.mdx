# Typography
At UQ, we have a set of complementary font assets and typeface rules that ensure a consistent look and feel of online content across devices and computers.

Typography is the art and technique of arranging type (letters and characters). It sets the theme and mood of the design via the typeface family and font styles like size, weight, style, colour, line-height, letter-spacing etc on single and multiple lines.

## Fonts

Online fonts are used for websites and applications, where the font can be embedded into code to display seamlessly. 

The 3 fonts used for UQ online applications are Roboto, Merriweather & Montserrat.

### Primary font

Roboto is a Google font and therefore can be displayed in online applications seamlessly. Roboto is our alternative to Gotham for online environments.

> While ***Gotham is our primary brand font***, We use alternative fonts online due to the costs to licence for websites and application and a lack of robust embedded versions. 

### Secondary font

MerriweatherMerriweather is a Google font and therefore can be displayed in online applications seamlessly. Merriweather can be used for pull out quotes and to break up long headings. It is ***not to be used for body text***.

## Loading fonts

Copy the HTML or CSS lines below and paste it into your project to load design system fonts.

### HTML
```html
<link rel="stylesheet" type="text/css" href="https://static.uq.net.au/v13/fonts/Roboto/roboto.css" />
<link rel="stylesheet" type="text/css" href="https://static.uq.net.au/v13/fonts/Merriweather/merriweather.css" />
<link rel="stylesheet" type="text/css" href="https://static.uq.net.au/v13/fonts/Montserrat/montserrat.css" />
```

### CSS/Sass
```css
@import url('https://static.uq.net.au/v13/fonts/Roboto/roboto.css');
@import url('https://static.uq.net.au/v13/fonts/Merriweather/merriweather.css');
@import url('https://static.uq.net.au/v13/fonts/Montserrat/montserrat.css');
```
