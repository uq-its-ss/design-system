# Grid menu

<StatusBadge.Review /><StatusBadge.DraftDoc />

## Overview

Grid menu is a list of unordered list elements that can be displayed in either 1, 2, 3 or 4 columns. 
By default it is displayed as a 3 column layout and changes on different responsive breakpoints.

The height of the grid menu items is by default equalised on its row.
