## Overview

The button component is used to navigate to a web link or to an anchor name tag on the same page or different page.

- The button categories are primary, secondary, secondary-light and purple.
- Each category has it's own variant styles like outline, text and expanded options to be used in different scenarios.

## Default example

### Primary

Primary buttons are used for primary actions on a webpage. For example, submit button on a form or other critical actions in a user workflow.
It should only appear once on a webpage.

### States

Default - Button text is not underlined. <br />
Hover - Button text is underlined. The background and text color may change depending on the variant.

### Breakpoints

Breakpoints have no effect on the button component and visually remains the same.

## Accessibility
