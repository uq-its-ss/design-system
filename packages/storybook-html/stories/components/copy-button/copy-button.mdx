## Overview

A button that allows copying of a string of text to the users clipboard.

## How to use

Implement the button markup following the example above, then call the copyButton() function as so:

```js
import { copyButton } from "@uqds/button";

const element = document.querySelector(".uq-copy-button");
const stringToCopy = element.dataset.copy;
copyButton(element, stringToCopy);
```
