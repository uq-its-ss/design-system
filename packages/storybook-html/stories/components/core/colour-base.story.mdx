# Base colour palette

The base colours are used to create all colour variations throughout the design system. Always start with a base colour when you need a colour that is not already defined in the system.

Use variable names over HEX or RGB values to ensure consistency can be maintained.

## UQ brand colours

These are the core brand colours for the University of Queensland. Refer to the brand guidelines for more information about how they are to be used.
