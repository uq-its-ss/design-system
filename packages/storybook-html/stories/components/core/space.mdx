## Layout spacing

Use this layout scale for arranging components and other UI parts into a full page layout. It has larger increments that are used to control the density of a design. Use the smaller layout tokens to create more dense compositions and direct relationships. Use the larger tokens to increase the amount of white space and to disassociate sections.

## Applying spacing

Consideration should be given to how spacing is applied within components and the page.

### Container space

Applies padding to containers

- For all user interface containers.
- Apply padding on all sides.

### Stacked spacing

Stacks content vertically, applying top margin.

- allows us to target the objects space through the css cascade, and the space to the next object.
- use for all stacked content. So for example panels, form fields and anything else that is stacked vertically.
