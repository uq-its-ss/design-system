## Overview

The `.uq-hero-basic` element provides an alternative hero.

### How-to

There are 4 elements within the basic hero.

- Title with "eyebrow heading": `h1.uq-hero-base__title` and `h1.uq-hero-basic__title .uq-hero-basic__eyebrow`
- Back button: `.uq-hero-basic__back`
- Tabs: `.uq-hero-basic__tabs .uq-tabs`
- Image: `.uq-hero-basic__image`

There are no size or colour variations for the basic hero component.
