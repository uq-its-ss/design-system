## Overview

Pagination, also known as paging, is the process of dividing a document into discrete pages.

The default pager uses `<a>` tags, but can also use `<button>` tags.

## Usage

Pagination should adapt depending on the number of available pages.

### 5 or less pages

Buttons for all pages show.

If on the first page, the Previous button does not show.

<Canvas of={PaginationStories.PagesLessFirst} />

If on the last page, the Next button does not show.

<Canvas of={PaginationStories.PagesLessLast} />

If on any other page, both Previous and Next buttons show.

<Canvas of={PaginationStories.PagesLessMiddle} />

### 6 or more pages

Visible page buttons adapts depending on the current page.

If on the first or second pages, the first 4 pages show. An ellipsis shows before the last page.

Clicking the ellipsis shows the first invisible page.

<Canvas of={PaginationStories.PagesMoreFirst} />

If on the last or second last page, the last 4 pages show. An ellipsis shows before the first page.

Clicking the ellipsis shows the next invisible page.

<Canvas of={PaginationStories.PagesMoreLast} />

If on any other page one less and one more pages show, with an ellipsis before the first and last pages.

Clicking either ellipsis skips 3 pages forward or back from the current page.

<Canvas of={PaginationStories.PagesMoreMiddle} />
