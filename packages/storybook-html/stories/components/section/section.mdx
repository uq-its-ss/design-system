## Overview

The `.uq-section` component is a wrapper to use around other components.

It provides correct **section spacing** between sections by using
a responsive `96px` and `48px` `margin-bottom` for desktop and tablet/mobile respectively.

Additionally, it provides a uniform heading and description and can be used
around card grids and panes.

### How-to

Use the `.uq-section__header` element to provide a h2 heading and
description to introduce another set of components.

Implement the markup structure below for the header.

<Canvas>
  <Story of={SectionStories.Section} />
</Canvas>

```html
<div class="uq-section">
  <div class="uq-section__header">
    <h2 class="uq-section__title">Title</h2>
    <div class="uq-section__description">
      <p>Description</p>
    </div>
  </div>
  <!-- Section content markup -->
</div>
```
