# Sections

<StatusBadge.Stable />

<StatusBadge.StableDoc />

## Overview

Sections are a layout style that wrap content in a white or grey background with a range of width, alignment, and padding options.

## Default

The default section displays left-aligned content, medium top and bottom padding, on a white background contained to the page width.

<Canvas withSource="none">
  <Story id="components-sections--sections-default" />
</Canvas>

```html
<section class="uq-section" aria-label="section title">
  <div class="uq-section__container">
    <div class="uq-section__header">
      <h2 class="uq-section__title"><!--section title--></h2>
    </div>
    <div class="uq-section__content">
      <!--section content-->
    </div>
  </div>
</section>
```

## Variants

The variants can be used in isolation or combined. For example if you would like a section to display with centre-aligned content in a narrow container, add both of the variant classnames to the section.

### Padding

Default sections have medium top and bottom padding. The following variants provide alternate padding options.

**Top padding:** Set the amount of padding at the top of the section:

*   No top padding - `uq-section--padding-top-none`
*   Large top padding - `uq-section--padding-top-large`

**Bottom padding:** Set the amount of padding at the bottom of the section:

*   No bottom padding - `uq-section--padding-bottom-none`
*   Large bottom padding - `uq-section--padding-bottom-large`

## How and when to use

Sections are primarily used on landing pages to create strips of content stacked vertically down the page, while containing the content to the page width.

## Accessibility

Include an `aria-label` with a value that describes the section. Assistive technologies use the value to define the content of the section.

```html
<section class="uq-section" aria-label="section title">
  <div class="uq-section__container">
    <div class="uq-section__header">
      <h2 class="uq-section__title"><!--section title--></h2>
    </div>
    <div class="uq-section__content">
      <!--section content-->
    </div>
  </div>
</section>
```
