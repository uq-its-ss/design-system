## Overview

A button that allows copying of a string of text to the users clipboard.

## How to use

Implement the social share links markup following the example above, then call the socialShare() function as so:

```js
import { socialShare } from "@uqds/social";

const element = document.querySelector(".uq-social-share");
socialShare(element);
```
