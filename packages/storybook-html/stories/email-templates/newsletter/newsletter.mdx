## Overview

The `.uq-container` component provides the **page container** by:

- Centering horizontally,
- using the `1140px` max-width,
- and adding responsive `32px` and `24px` page gutters for desktop and tablet/mobile respectively.

### How-to

The `.uq-container` element can be used anywhere, for example nested within other components.

It must not be a descendant of other `.uq-container` elements or elements that use
`@include layout-containment`.
