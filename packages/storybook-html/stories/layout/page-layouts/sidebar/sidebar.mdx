## Overview

The `.uq-sidebar-layout` component provides the grid system used for the local
navigation.

### How-to

The element must first be wrapped in a `.uq-container` element to provide the
**page container**.

Then implement the component as per the HTML example below.

`.uq-section` elements are then used within `.uq-sidebar-layout__main` to
provide correct **section spacing**.

<Canvas of={SidebarStories.Sidebar} />

```html
<div class="uq-container">
  <div class="uq-sidebar-layout">
    <div class="uq-sidebar-layout__sidebar"></div>
    <div class="uq-sidebar-layout__main">
      <div class="uq-section"></div>
    </div>
  </div>
</div>
```
