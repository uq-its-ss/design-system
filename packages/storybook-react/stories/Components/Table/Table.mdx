# Table

<Description of={stories} />

## Installation

```bash
npm i @uqds/react
```

## Usage

```js
import { Table } from "@uqds/react";
```

The Table component is a simple wrapper around the HTML `<table>` element but
with appropriate CSS styles applied. It should be used in place of, and in the
same way as, the HTML `<table>`.

See the Table component in the HTML UQDS guide for more details on expected use.
