# Usage

To use `@uqds/react` components follow these steps.

## Installation

First install the React package.

```bash
npm i @uqds/react
```

## Global CSS

Include Global CSS once for a whole project. Component CSS is handled with
CSS Modules.

### PostCSS

```css
@import "@uqds/react";
```

### JS

```js
import "@uqds/react/dist/main.css";
```

## Component usage

Follow the docs page of each component for individual component usage.
