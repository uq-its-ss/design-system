# Usage

To use `@uqds/react` components follow these steps.

## Installation

First install the React package.

```bash
npm i @uqds/react
```

## Implementation

Import the CSS in a top-level file, such as `App.tsx`.

```js
import '@uqds/react/css';
```

In a React .tsx or .jsx file, import and use the components as needed.

```js
import { TextInput } from '@uqds/react';

<TextInput placeholder="Example" />
```

## Component usage

Follow the docs page of each component for individual component usage.
